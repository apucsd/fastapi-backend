# PROJECT_CONTEXT.md

## Project Name

ApplyFlow AI

## Goal

Build an AI-powered job application assistant that helps users discover relevant jobs, analyze job descriptions, generate personalized applications, and manage their application process.

This project is primarily for:

1. Learning FastAPI deeply
2. Learning AI integration in backend systems
3. Learning automation workflows
4. Building a portfolio project that demonstrates real-world engineering skills

## Core Idea

A user uploads their resume and profile information.

The system automatically:

* Finds relevant jobs
* Analyzes job descriptions
* Calculates job match scores
* Identifies missing skills
* Generates personalized application emails and cover letters
* Tracks application history

The user remains in control and approves applications before they are sent.

## Important Learning Areas

While building this project, focus on:

* FastAPI
* PostgreSQL
* AI integrations
* Background jobs and scheduling
* Email integrations
* Automation workflows
* Document processing
* Vector search and RAG (later)

## Project Philosophy

This is not another chatbot project.

The goal is to build a real product that solves a real problem and demonstrates backend engineering, AI integration, automation, and system design skills.

When making implementation decisions:

* Prefer learning value over shortcuts
* Prefer real-world architecture over tutorial architecture
* Prefer practical features over flashy AI features

## Future Vision

A user can upload a resume once and the platform can continuously:

* Search for jobs
* Evaluate opportunities
* Generate applications
* Provide career insights
* Track progress

acting as an AI-powered job search assistant.
